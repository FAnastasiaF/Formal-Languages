# Практикум №1

#### Задача:
*(18 вариант)*

На ввод подается язык из символов из {a, b, c, 1, ., +, *}. Даны α и буква x. Найти максимальное k, такое что в L есть слова, начинающиеся с x<sup>k</sup>.

#### Алгоритм:

Считываем регулярное выражение и букву x. Проходимся по каждому элементу и запихиваем информацию<sup>о</sup> о нём в стек, 
если этот элемент буква.  
А если это операция, то извлекаем нужное количество элементов из стека(2 элемента для "+" и "." и 1 для "*"), 
делаем с ними эти операции и возвращаем на стек результат. Данный алгоритм работает за O(длины заданного регулярного выражения).

Информация<sup>о</sup>    
1. Тип символа:  
0 - не только из x  
1 - только из x   
2 -  есть вариант со всеми x(или 1) и есть вариант не со всеми x
2. Количество букв x, в слове только из x:
0 - если тип символа 0
3. Количество букв x, в слове не только из x:
0 - если тип символа 1
4. Начинается ли это слово на x*
